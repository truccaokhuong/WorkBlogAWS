### User Experience Flow

This section demonstrates the main actions of end users in Travel Platform: accessing the home page, searching for places, viewing place details, saving favorite places, creating trips, managing itineraries, and tracking booking status.

#### 1. Home Page

Users access the home page to view the overall interface and start searching for travel destinations.

![Home page for visitors](/images/demo/demo-01-home.png)

#### 2. Place Search

Users can search for places by keyword, for example searching for destinations in Da Lat.

![Visitor searches for places in Da Lat](/images/demo/demo-02-search-dalat.png)

#### 3. Place Detail

After selecting a place, the system displays details such as images, description, reviews, and related information.

![Place detail page](/images/demo/demo-03-place-detail.png)

#### 4. Save Favorite Place

Authenticated users can save favorite places for later trip planning.

![User saves a favorite place](/images/demo/demo-04-save-place.png)

#### 5. Saved Places

Saved places are collected in the Saved Places screen, allowing users to review and choose places for their itinerary.

![User saved places list](/images/demo/demo-05-saved-places.png)

#### 6. Create New Trip

Users enter trip information such as name, date range, and description to start planning an itinerary.

![User enters information for a new trip](/images/demo/demo-06-create-trip.png)

#### 7. Day-based Itinerary

Places in a trip are organized by day and visit order, making the itinerary easier to follow.

![Itinerary organized by day and visit order](/images/demo/demo-07-itinerary-by-day.png)

#### 8. Submit Booking Request

Users submit a booking request and receive a result from the system.

![Booking request result](/images/demo/demo-09-booking-result.png)

#### 9. Track Booking Status

After creating a booking, users can track the booking status in the system.

![User tracks booking status](/images/demo/demo-10-booking-status.png)
