This section lists the AWS blog posts I have translated and studied during my internship:

###  [Blog 1 - Amazon Bedrock – Generative AI Service on AWS](3.1-Blog1/)
An in-depth exploration of Amazon Bedrock, AWS's managed generative AI service. Covers the architecture of AI applications using API Gateway, Lambda, and Bedrock, integration with multiple Foundation Models (Titan, Claude, Llama, Cohere, Mistral), RAG patterns with Knowledge Bases, and CloudWatch monitoring.

###  [Blog 2 - AWS Well-Architected Framework in Cloud System Design](3.2-Blog2/)
A comprehensive breakdown of the six pillars of AWS Well-Architected Framework: Operational Excellence, Security, Reliability, Performance Efficiency, Cost Optimization, and Sustainability. Explains how each pillar contributes to building robust cloud architectures.

###  [Blog 3 - Survey of Amazon S3 Storage Classes and Selection Criteria](3.3-Blog3/)
A detailed comparison of all Amazon S3 Storage Classes: Standard, Intelligent-Tiering, Standard-IA, One Zone-IA, Glacier Instant Retrieval, Glacier Flexible Retrieval, and Glacier Deep Archive. Provides guidance on choosing the right storage class based on access frequency, retrieval time, cost, availability, and retention duration.
